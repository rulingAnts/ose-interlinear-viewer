Opening in Excel

  1. Download the .tsv file using the button above.
  2. Open Excel and choose File › Open, then select the downloaded file. If Excel doesn't show .tsv files, change the file type filter to All Files.
  3. The Text Import Wizard will open. Choose Delimited and click Next.
  4. Under Delimiters, check Tab (uncheck everything else) and click Finish.
  5. Each vernacular word and its gloss will be in its own column, aligned vertically.

In Google Sheets, go to File › Import, upload the .tsv file, choose Tab as the separator, and click Import.
